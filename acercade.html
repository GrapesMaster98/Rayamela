¿Que es Rayamela?

Rayamela es una pagina totalmente rídicula creada por GrapesCode (Oliver Klassen) como una broma y de paso practicar el uso de JavaScript con HTML de manera mas avanzada.

El proyecto es open source (código abierto), puedes ver todo el HTML y el JavaScript de esta pagina sin problemas dando click en la parte de "código abierto" ahí arriba, asegurate de leer la ventana de inicio.
